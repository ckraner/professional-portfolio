# Shiny App

This contains the indivudual files for the stats.explor.r app that I created during my 700 level course. It is in seperate files because in the early days of shiny, you couldn't make consistent packages.

Please note: This program will not run from these files. Additional packages I wrote were not included. Modification is needed with the new Shiny system. And additionally, many features are now native to Shiny. I was an early adoptor.

For an example of this app running, please see artifacts.

## Contents
 - stats.explor.r - Main program
 - label.explor.r - Data wrangling program
 - installer.r - Installer since shiny packages weren't a thing. Creates directorys and uses cat to ouptput the required files
 - files - Here is a folder that contains the structure that the installer program puts in.
