## 📘 COA‑IIIA Validation (Springer, 2023)
**Educational Assessment, Evaluation and Accountability**  
**Authors:** Reeves, Onder, Kraner

### Summary
Psychometric evaluation of the Conceptions of Assessment III‑Abridged (COA‑IIIA) among U.S. pre‑service and in‑service teachers.

### Key Points
- Tested 16 prior COA‑IIIA models using CFA.  
- Identified best‑fitting structure (Model 6c).  
- Demonstrated metric invariance across teacher groups.  
- Provided reliability evidence for retained factors.

### My Role
- Conducted all statistical analyses (CFA, invariance, reliability).  
- Wrote the technical analysis documents used to build the published manuscript.  
- Generated model comparisons and fit diagnostics that shaped the final paper.

## 👓 Google Glass Instant Feedback System (IEEE ICALT, 2022)
**International Conference on Advanced Learning Technologies**  
**Authors:** Tscholl, Demir, Kraner

### Summary
Prototype system using Google Glass to deliver real‑time, private feedback during student presentations.

### Key Points
- Provided instant cues (e.g., *Speed up*, *Eye contact*, *Louder*).  
- Students reported high usability and increased self‑efficacy.  
- Identified design improvements for future iterations.

### My Role
- Technical onboarding: learned the device and created usable workflows.  
- Instructional enablement: taught participants how to use the Glass system.  
- System integration: ensured the device and interface were functional for study deployment.  
- Did not conduct interviews or sentiment analysis; authorship reflects technical contributions.

## 🔊 HVAC Acoustic Propagation Model (Personal Technical Project)

### Summary
Self‑directed physics and engineering project modeling outdoor HVAC sound propagation using wave‑based acoustics.

### Key Points
- Developed a frequency‑domain acoustic model for single‑unit, line‑array, and square‑array HVAC configurations.  
- Derived behavior for Fresnel zones, constructive/destructive interference, and ground‑reflection phase interactions.  
- Produced a structured technical paper detailing the theoretical framework and mapping each component to future Python implementation.  
- Designed the model to be modular for future simulation or program‑evaluation workflows.

### My Role
- Sole author of all derivations, modeling decisions, and technical documentation.  
- Built the mathematical framework and engineering assumptions from scratch.  
- Wrote the full analysis paper outlining implementation steps.
